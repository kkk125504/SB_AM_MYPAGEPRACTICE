### 게시물 수정기능 구현

```java
  	@RequestMapping("usr/article/doModify")
    @ResponseBody
    public Object doModify(int id, String title, String body) {
            Article article = getArticle(id);	
        if (article == null) {
            return id + "번 게시물은 존재하지 않습니다.";
        }

        modifyArticle(id, title, body);
        return article;
    }
```

- 게시물이 없을때 String으로 return, 게시물이 존재한다면 게시물을 return 하고자 한다.
- doModify()메서드의 return 타입을 String으로 하면 위에 if문에서의 return과 if문을 타지않을때 return 타입을 String 으로 맞춰야한다.
- 하지만 return타입이 Object라면 모든 타입을 return 할수있게 된다.

```java
    private void modifyArticle(int id, String title, String body) {
        Article article = getArticle(id);
        article.setTitle(title);
        article.setBody(body);
    }
```

- set()메서드를 이용하여 private 필드에 간접적으로 접근
